# AI-Based Speech Therapy Exercise Recommendation System (Level 1)

## Project Overview
This project recommends suitable speech therapy exercises for children using Machine Learning and Deep Learning techniques. The system analyzes speech-related features and predicts the most appropriate therapy exercise.

---

## Features
- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Machine Learning Model
- Deep Learning (Artificial Neural Network)
- Speech Therapy Exercise Prediction

---

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- TensorFlow
- Keras
- Matplotlib
- Jupyter Notebook

---

## Project Workflow

1. Load Dataset
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Encoding
5. Train Machine Learning Model
6. Train Deep Learning Model
7. Evaluate Model
8. Predict Recommended Exercise

---

## Results
- Successfully trained ML and DL models.
- Achieved high prediction accuracy.
- Generated personalized speech therapy exercise recommendations.
